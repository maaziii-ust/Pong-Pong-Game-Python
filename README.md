# Pong Pong Game (Python + Pygame)

## Overview

Pong Pong Game is a graphical two-player arcade game developed in Python using the Pygame library. Players control paddles to bounce the ball and score points against their opponent.

## Features

* Two Player Gameplay
* Real-Time Ball Movement
* Paddle Controls
* Collision Detection
* Score Tracking
* Smooth Animation
* Graphical User Interface

## Technologies Used

* Python 3.12
* Pygame

## Controls

### Left Player

* W → Move Up
* S → Move Down

### Right Player

* ↑ → Move Up
* ↓ → Move Down

## Installation

Install Pygame:

```bash
pip install pygame
```

Run the game:

```bash
python pong_game.py
```
